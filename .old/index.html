How is GPA Calculated ?

In order to calculate GPA, we would first need the following 3 components:

  1. Total Credits: Total Credits earned over all the modules taken
  2. Module Grade: Individual grade of all the modules you have taken
  3. Module Credit: Individual credit of all the module you have taken
